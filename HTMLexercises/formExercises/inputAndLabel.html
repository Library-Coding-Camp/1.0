Input element

The input tag is used to make a textbox and other input related fields. The input tag is a self-closing tag (meaning it does not have a closing tag) and has one (main) attribute, the type attribute. For now, we will set this attribute to text, since we are expecting a text input.
An HTML text box looks like this:



It is arguably the ugliest default HTML elements. Thankfully, CSS can be used to style it.
The HTML to create a textbox is the following:

<input type = "text">

As an exercise, create two textboxes, one for email and one for password. As an extra challenge, on the password field try to change the type attribute to "password" and see the difference when you attempt to type in it.
