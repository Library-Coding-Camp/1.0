Label Element

As you know, the input element creates a text box; however, there is no way to label it. While you can use a paragraph element, HTML has a specific tag that specializes in labeling input elements. It is called the label element. The label element has an attribute called 'for' This attribute's value (what it is equal to) is the same as the input element's id. (An id is like class, except it is specific to an element. Ids and classes will be discussed in more detail later).

<label>Label of Input element
<input type = "text">
</label>
