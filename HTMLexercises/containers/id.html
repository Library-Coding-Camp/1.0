Id

An id is an attribute that is unique to an HTML element and can be added to any HTML element. It is also used to give styles to an object and, more commonly, manipulate objects in Javascript. We will not go over adding styles to an id in this lesson, but we can go over how to add an id to an element. For example, the h1 tag tag below had the id ‘title’.

<h1 id = “title”></h1>

Note that the id name (id = “idName”) cannot have spaces.
As an exercise, try to create a textbox with the type 'password' and an id of 'password'.
