Div Element

The div element is an HTML element used to ‘contain’ or ‘encompass’ other HTML elements. This allows our HTML elements to be more organized. For example, say we were making a title with a subtitle. With the div element it would look like:

<div class = “title”>
    <h1 class = “title-Large”>Colors</h1>
    <h2 class = “title-Small>Red</h2>
</div>
You can also have div elements in div elements, for example:
<div class = “title”>
    <div>
         <h1 class = “title-Large”>Colors</h1>
    </div>
    <div>
         <h2 class = “title-Small>Red</h2>
    </div>
</div>

The div element will also be useful in styling, since an element with a particular style passes down its styles to the elements inside it (also known as its child elements).
As an exercise, try to create a div element with the class "divElement" and id "divElem" with two other divs inside it.
