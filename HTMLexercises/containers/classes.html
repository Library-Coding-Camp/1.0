Classes

A class is an attribute that can be applied to any element. It is used to group elements with similar styles together. It is also used to manipulate HTML elements in Javascript. While we will not go over how to add styles to classes yet, we can go over how to add a class to an HTML element. For example: the p tag below has the class name ‘red-line’.

<p class = “red-line”> </p>

Please note the class name (class = “className”), can be any text you want it to be, but cannot have spaces.
As an exercise, try to create an h1 tag with the class 'title' and an h3 tag under it with the class name 'subtitle'.
