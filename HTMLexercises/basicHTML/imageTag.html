Image Tag

To add images to a webpage, we use the image tag. It looks like this:

<img src=”image address”>

However, images do not always load. When they do not, it is always best to have a short description where the image should be (less than a sentence). You can do this by adding “alt = image description” in the image tag. For example: <img src = “image address” alt = “description”>

As an exercise, add an image of a cat with an alt attribute of your choice.
