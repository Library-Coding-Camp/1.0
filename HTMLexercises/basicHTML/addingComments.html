HTML Comments

Comments are lines that are not read by the computer, but are for humans to help them understand the HTML code. You can comment out HTML using the following syntax:

<!-- This is a comment! -->

Keep in mind a comment can continue for several lines

<!--
This is also a comment!
-->

As an exercise, write an h1 tag with the content "a heading", and a comment under it that says "This is the heading of the page"
