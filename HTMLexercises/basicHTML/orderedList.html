Ordered List

An ordered list looks the exact same as an unordered or bulleted list, except for the fact it is declared with a ‘ol’ tag instead of a ‘ul’ tag. Here is the example HTML to create an ordered list.

<ol>
    <li>Number 1</li>
    <li>Number 2</li>
</ol>

This produces the output:
    1. Number 1
    2. Number 2

As an exercise, try to create an ordered shopping list with the title "Shopping list". CLUE: you can't create a heading for a list using a ul tag, try using one of the heading tags.
