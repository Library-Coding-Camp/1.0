HTML Tag Structure

Almost every HTML tag has an opening tag and a closing tag, with some content in the middle. (<TagName>content<\TagName>). As an example, we will look at the h1 tag (which will be explained in greater depth in the next lesson).

<h1>Hello World</h1>

This will print the text “Hello World” on the website.

The opening tag is <h1>, and the closing tag is </h1>. Note how the tags are the same aside from the \ in front of the tagname of the closing tag. An opening tag, closing tag and the content in between them make up a HTML element.
