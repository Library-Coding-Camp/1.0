HTML Headings

The heading tags are as follows:

<h1></h1>
<h2></h2>
<h3></h3>
<h4></h4>
<h5></h5>
<h6></h6>

These create headings for a webpage. The h2 element will be smaller than the h1 heading (and thus should be used as a subheading), the h3 element will be smaller than the h2 element, and so on. The h1 tag should be used as the main heading or title of the website
